# Token-Reduction Benchmark — RTK × Caveman × Semantic

- Tasks: **8**  |  Tokenizer: **gpt-tokenizer (o200k_base — approximates Anthropic)**
- RTK transform mode: **heuristic** (set `RTK_BENCH_BIN` to use the real binary)
- Baseline (all features off) mean total: **2930.1 tokens/task**

## Main effects (average % reduction when a flag is turned ON)

| Feature | Mean reduction | ± std | Consistent? |
|---------|---------------:|------:|:-----------:|
| rtk | 23.6% | 9.4% | yes |
| caveman | 6.1% | 3.0% | yes |
| semantic | 61.4% | 4.8% | yes |

## Conclusion

- **rtk**: **consistently reduces** tokens (~23.6% ± 9.4).
- **caveman**: **consistently reduces** tokens (~6.1% ± 3.0).
- **semantic**: **consistently reduces** tokens (~61.4% ± 4.8).

- All three together: **73.0%** fewer tokens than baseline (2930.1 → 792.1 tokens/task).

> Note: token counts are an offline estimate; the comparison between combinations is the signal, not the absolute figure.
